O distributie server JackRabbit este formata din:
- JAR-ul JackRabbit standalone
- XML-urile de configurare
- Fisierul de comanda pentru pornirea server-ului

Fisierele necesare se gasesc in "Server JackRabbit.zip" (se va extrage intr-un folder ce va fi considerat radacina
server-ului) si in sursele proiectului.

In continuare, vom considera radacina server-ului "SERVER_HOME".

JAR-ul JackRabbit standalone:
	- Acesta trebuie sa contina si clasele driver-ului JDBC daca se va folosi o baza de date ca persistenta.
	- Fisierul curent, "jackrabbit-standalone-1.5.7.jar", contine si driver-ul JDBC pt. Oracle.
	- Fisierul trebuie tinut in "SERVER_HOME".

XML-urile de configurare:
	- Cele doua fisiere de configurare sunt "repository.xml" si "indexing_configuration.xml".
	- Cele doua fisiere de configurare se gasesc in sursele proiectului.
	- Cele doua fisiere trebuie tinute in "SERVER_HOME/jackrabbit".

Fisierul de comanda pentru pornirea server-ului:
	- Este un fisier de tip CMD / SH ce porneste server-ul.
	- Exista in sursele proiectului variante pentru Windows ("start_server_Windows*.cmd")
	si pentru Unix ("start_server_Unix*.cmd").
